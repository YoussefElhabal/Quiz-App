# 🎯 Quiz App

A modern, responsive Quiz Application built with **HTML**, **CSS**, and **JavaScript**. The app allows users to customize quiz settings, answer multiple-choice and true/false questions, view their score in real time, and restart the quiz at any time.

---

## 🚀 Features

* ✅ Built using **JavaScript ES6+**, **OOP**, and **JS Modules**
* 📱 Fully **responsive** and mobile-friendly
* 🎮 User-friendly interface with real-time feedback
* 🧠 Supports **Multiple Choice** and **True/False** question types
* ⚙️ Users can select:
    * Question **category**
    * **Difficulty** level (easy, medium, hard)
    * **Number** of questions
* 📊 Live score tracking during the quiz
* 🔁 Option to **restart** the quiz after completion

---

## 🛠️ Technologies Used

* **HTML5** – Semantic and accessible structure
* **CSS3** – Responsive design with modern styling
* **JavaScript (ES6+)** – Logic, interactivity, and API integration

    * ES6 Classes
    * Modular architecture (JS Modules)
    * Fetch API (if you're using it to get questions from an external source)

---

## 🎮 How to Use

1. Open the app in your browser.
2. Select:
    * **Category**
    * **Difficulty level**
    * **Number of questions**
3. Click **Start Quiz**.
4. Answer each question.
5. See your **current score** after each question.
6. At the end, view your **total score** and click **Try Again** to start a new quiz.
